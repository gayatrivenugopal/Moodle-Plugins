moodle-local_accesscontrol
====================
A local plugin that allows admin to allow certain users to bypass module level restrictions.

Version
-------
1.1.0 (2014063001)

Requires:
--------
Moodle 2.0 or higher

Installation:
------------
Download zip from: https://github.com/innovg/moodle-local_accesscontrol/tree/master/accesscontrol
Unzip into the 'local' subfolder of your Moodle install.
Rename the new folder to accesscontrol.
Visit http://yoursite.com/admin to finish the installation. 

Documentation:
-------------
Please feel free to contribute documentation in the relevant area of
the MoodleDocs wiki.
